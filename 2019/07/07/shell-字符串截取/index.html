字符串的截取

%号截取，删除右边字符，保留左边字符

%t*表示从右边开始，删除第一个 t及右边的字符

%%m* 表示从右边开始，删除最后（最左边）一个t及右边的字符

#号截取，删除左边字符，保留右边字符。

#*t表示从左边开始，删除第一个 t及左边的字符

##*t表示从左边开始删除最后（最右边）一个t及左边的所有字符

切片截取

从左边第几个字符开始，总字符的个数

从第一个字符开始，总个数为4

从左边第几个字符开始，直到结束

从第4个开始，到结束

从右边第几个开始，总字符个数

从右边第三个开始算起，向右2个字符

从右边第几个字符开始，直到结束

从右边第三个开始，向右直到结束

例子：我需要将简书上的文章发布到网站上，我已经下载好了，可是因为有点多，需要每次都手动去一个一个的发布，太麻烦了，所以就想有什么简单的方法可以解决。
