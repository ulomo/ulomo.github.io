迭代器

iterate迭代实现了__iter__内置特殊方法

使用迭代器与列表的区别：

  1. 可能只需要返回下一个值就可以了，并不需要一次获取所有数据

  2. 列表一次获取所有值占用内存高，而迭代器一次返回一个值，占用内存低

  3. 实现了__iter__的对象是可迭代的，实现了__next__的对象是迭代器。

从迭代器创建序列

虽然可以使用序列，但是迭代器其实也可以创建序列
