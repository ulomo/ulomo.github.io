虚拟机sharefolder

秋日摘桂花做饼
2019-04-27
linux › archlinux
系统
  • 安装插件insert Guest Additions cd image

  • 会提示添加盘符，新建一个空的option device， 再安装就可以了

  • 在~/virtualbox vms目录下建一个share_folder文件夹，用来在linux和windows上共享文件

  • 在虚拟机上machine-settings-shared folders-新建一个共享目录，路径为之前设置的。选择自动挂载

  • 然后就可以在虚拟机中看到挂载的虚拟磁盘了，通过这个共享磁盘实现共享文件。
