linux-jdk1-6的安装步骤

秋日摘桂花做饼
2019-05-02
java

  1. 首先下载jdk1.6的二进制文件(可以直接安装，不要解压)
    链接：https://pan.baidu.com/s/19lcBWUeYoM3kgaJQ-ybsrQ
    提取码：frd9

  2. 然后通过xshell将jdk传输到linux中

    rz

  3. 在linux中创建jdk安装目录

    sudo mkdir /usr/java

  4. 复制jdk到/usr/java目录下

    sudo cp ~/jdk-6u45-linux-x64.bin /usr/java
cd /usr/java
ls

  5. 修改jdk文件权限

    sudo chmod 775 jdk-6u45-linux-x64.bin

  6. 安装jdk

    ./jdk-6u45-linux-x64.bin

  7. 配置jdk环境变量

    sudo vim /etc/profile

    添加

    JAVA_HOME=/usr/java/jdk1.6.0_45
CLASSPATH=.:$JAVA_HOME/lib/tools.jar
PATH=$PATH:$JAVA_HOME/bin
export JAVA_HOME CLASSPATH PATH

  8. 刷新一下，使配置环境变量生效

    source /etc/profile

  9. 查看是否安装成功

    java -version

1. 以上步骤需要对照自己安装目录这些什么的进行修改，如果操作步骤和我一样可以直接复制
2. 另外如果提示那条命令权限不足，就在命令前面添sudo
