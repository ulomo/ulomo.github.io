xmonad以及虚拟机全屏的问题

秋日摘桂花做饼
2020-05-21
linux › archlinux
linux
  • xmonad编译报错

安装的archlinx虚拟机中的xmonad在编译时报错，所有的包都无法找到。这个问题之前在我的archlinux遇到过，是一次更新引起的，解决办法就是重新加载数据库

但是这个方法用在虚拟机中没有效果。目前还没找到解决办法

  • 虚拟机全屏

有的虚拟机例如win10和kali都可以在挂载guest.iso后成功运行，并实现全屏。但是像manjaro和自己安装的archllinux却无法做到。

安装这个guest插件有两种方式：

  1. 安装在host上，然后挂载到虚拟机，在虚拟机中运行这个挂载的文件
  2. 直接安装在guest中

之前我一直都是使用的第一种方法，但是有的挂载就会报错，所以找到了第二种方法

安装这两个就好了，然后在guest中设置一下分辨率就可以设置全屏了。
